Decision Making
================

    Decision making allows a program to choose different actions based on certain conditions.
   It helps your program make choices â€” just like humans do.

   ğŸ”¹ Example:

        if temperature > 30:
            print("Itâ€™s hot outside!")
        else:
            print("Nice weather!")

    Types of Decision-Making Statements in Python
    -----------------------------------------------

        1.if statement

        2.if...else statement

        3.if...elif...else statement

        4.match...case statement (introduced in Python 3.10)


        ğŸ§© 1. if Statement

            âœ… Used to execute a block of code only if a condition is True.

            ğŸ”¹ Syntax:
                
                if condition:
                    statement(s)
            
            ğŸ”¹ Example:

                age = 18

                if age >= 18:

                    print("You are eligible to vote.")

        
        ğŸ§© 2. if...else Statement

            âœ… Used when we want one block of code to run if the condition is True, and another block if itâ€™s False.

            ğŸ”¹ Syntax:

                    if condition:
                        statement(s)
                    else:
                        statement(s)
            
            ğŸ”¹ Example:

                number = 5

                if number % 2 == 0:

                    print("Even number")

                else:

                    print("Odd number")
        
        
        ğŸ§© 3. if...elif...else Statement

            âœ… Used when you need to check multiple conditions.

            ğŸ”¹ Syntax:
                    if condition1:
                        statement(s)
                    elif condition2:
                        statement(s)
                    elif condition3:
                        statement(s)
                    else:
                        statement(s)
            
            ğŸ”¹ Example:

                    marks = 85

                    if marks >= 90:
                        print("Grade A+")
                    elif marks >= 75:
                        print("Grade A")
                    elif marks >= 60:
                        print("Grade B")
                    else:
                        print("Grade C")
            ğŸŸ¢ Note:

                Only the first True condition is executed.

                The else block is optional.

        ğŸ§© 4. match...case Statement (Python 3.10+)

            âœ… Used for pattern matching â€” similar to switch in other languages.

            ğŸ”¹ Syntax:
                    match variable:
                    case value1:
                        statement(s)
                    case value2:
                        statement(s)
                    case _:
                        statement(s)   # default case

            ğŸ”¹ Example:

                    day = 3
                    match day:
                        case 1:
                            print("Monday")
                        case 2:
                            print("Tuesday")
                        case 3:
                            print("Wednesday")
                        case 4:
                            print("Thursday")
                        case 5:
                            print("Friday")
                        case 6:
                            print("Saturday")
                        case 7:
                            print("Sunday")
                        case _:
                            print("Invalid day")

            ğŸŸ¢ Notes:

                The underscore _ acts as a default case (similar to else).

                You can also combine multiple cases:

                match day:
                    case 1 | 2 | 3 | 4 | 5:

                        print("Weekday")

                    case 6 | 7:
                        print("Weekend")


ğŸ” Looping in Python
=======================

    Looping allows you to execute a block of code repeatedly as long as a certain condition is true or for a fixed number of iterations.

    It helps reduce repetition and makes programs more efficient.


    ğŸŒ± Why Loops?

        Without loops, we would need to write the same code multiple times.

        ğŸ”¹ Example (without loop):

            print("Hello")
            print("Hello")
            print("Hello")
            print("Hello")
            print("Hello")

        ğŸ”¹ Example (with loop):

            for i in range(5):
                print("Hello")

        ğŸŒ€ Types of Loops in Python

            | Type of Loop   | Description                                                           |
            | -------------- | --------------------------------------------------------------------- |
            | **while loop** | Repeats a block of code **as long as a condition is True**.           |
            | **for loop**   | Iterates over a **sequence** (like list, string, tuple, range, etc.). |

        
        ğŸ§© 1. while Loop

            âœ… Used when you donâ€™t know in advance how many times you want to repeat â€” loop continues until a condition becomes False.

            ğŸ”¹ Syntax:

                while condition:
                    statement(s)

            ğŸ”¹ Example:

                i = 1
                while i <= 5:
                    print("Count:", i)
                    i += 1

            ğŸ§  Important Notes:

                . The condition is checked before every iteration.

                . If the condition never becomes False, it leads to an infinite loop.

                . Always make sure to update your loop variable inside the loop.

        ğŸ§© 2. for Loop

            âœ… Used when you know how many times to iterate â€” or when looping through a sequence (like list, string, range, etc.).

            ğŸ”¹ Syntax:

                for variable in sequence:
                    
                    statement(s)

            ğŸ”¹ Example:

                for i in range(5):

                    print("Value:", i)

            ğŸ”¹ Example: Looping Over a List

                fruits = ["apple", "banana", "cherry"]

                for fruit in fruits:

                    print(fruit)

            ğŸ”¹ Example: Looping Through a String

                for ch in "Python":
                    print(ch)

            ğŸ§® Example: Using range()

                The range() function generates a sequence of numbers.

                | Syntax                     | Description       | Example         | Output    |
                | -------------------------- | ----------------- | --------------- | --------- |
                | `range(n)`                 | 0 to n-1          | `range(5)`      | 0,1,2,3,4 |
                | `range(start, stop)`       | start to stop-1   | `range(2,6)`    | 2,3,4,5   |
                | `range(start, stop, step)` | Increment by step | `range(1,10,2)` | 1,3,5,7,9 |

            
            ğŸ”¹ Example: Print Even Numbers from 2 to 10

                for i in range(2, 11, 2):
                    print(i)

        ğŸ§± Loop Control Statements

            used to change the normal flow of loop execution.

            | Statement  | Description                                         |
            | ---------- | --------------------------------------------------- |
            | `break`    | Exits the loop immediately                          |
            | `continue` | Skips the current iteration and continues with next |
            | `pass`     | Does nothing (used as a placeholder)                |

ğŸ Python Functions
====================

    1ï¸âƒ£ What is a Function?

        A function is a block of organized, reusable code that performs a specific task.

        It helps you avoid repeating the same code multiple times â€” you write it once and use it whenever needed.


    ğŸŒ± Why Use Functions?

        . Makes code modular (divides big problems into smaller parts).

        . Improves code reuse (use same function many times).

        . Enhances readability and debugging.

        . Saves time and effort when making changes.

    ğŸ§© 2ï¸âƒ£ Types of Functions in Python

        | Type                       | Description               | Example                                |
        | -------------------------- | ------------------------- | -------------------------------------- |
        | **Built-in Functions**     | Predefined in Python      | `print()`, `len()`, `input()`, `max()` |
        | **User-defined Functions** | Created by the programmer | Custom functions like `def greet():`   |


    âš™ï¸ 3ï¸âƒ£ Defining and Calling a Function

        def function_name():
            # block of code

        function_name() #calling funtion


    ğŸ”¹ Example:

        def greet():
            print("Hello! Welcome to Python Functions.")

        greet()
        greet()

    ğŸ§® 4ï¸âƒ£ Function with Parameters

        Syntax

            def function_name(parameter1, parameter2, ...):
                # code block

        âœ… Example:

            def greet_user(name):
                print("Hello", name, "Welcome to Python!")

            greet_user("vipin")
            greet_user("abin")


    ğŸ¯ 5ï¸âƒ£ Function with Return Value


        âœ… Syntax:

            def function_name():
                return value

        âœ… Example:

            def add(a, b):
                
                return a + b

            result = add(5, 3)
            print("Sum is:", result)

    ğŸ“¦ 6ï¸âƒ£ Parameters vs Arguments

        | Term          | Meaning                                        |
        | ------------- | ---------------------------------------------- |
        | **Parameter** | Variable listed inside function definition.    |
        | **Argument**  | Actual value passed when calling the function. |

    
       ğŸ”¹ Example:

                def square(num):   # num â†’ parameter
                    return num ** 2

                print(square(4))   # 4 â†’ argument

    ğŸ§  7ï¸âƒ£ Types of Arguments in Python

            | Type                          | Description                          | Example                   |
            | ----------------------------- | ------------------------------------ | ------------------------- |
            | **Positional Arguments**      | Values passed in correct order       | `add(10, 20)`             |
            | **Keyword Arguments**         | Specify parameter name               | `add(a=10, b=20)`         |
            | **Default Arguments**         | Have a default value if not provided | `def greet(name="Guest")` |
            | **Variable-length Arguments** | Accept multiple values               | `*args`, `**kwargs`       |

    
    ğŸ§© 8ï¸âƒ£ Default Argument Example

        def greet(name="Guest"):
            print("Hello", name)

        greet("hari")
        greet()      # uses default value "Guest"

    
    ğŸ§© 9ï¸âƒ£ Variable-length Arguments

        âœ… Using *args â†’ for multiple positional arguments

        Example

            def add_numbers(*numbers):
                total = 0
                for num in numbers:
                    total += num
                return total

            print(add_numbers(2, 3, 4))


    âœ… Using **kwargs â†’ for multiple keyword arguments

        def show_details(**info):
            for key, value in info.items():
                print(key, ":", value)

        show_details(name="hari", age=25, city="Kochi")

    
    ğŸ§   Anonymous (Lambda) Functions

        A lambda function is a short, one-line function without a name.

        âœ… Syntax:

            lambda arguments: expression
        
        âœ… Example:

            square = lambda x: x * x
            print(square(5))   # Output: 25

    ğŸ”š Summary

        | Concept             | Description                             |
        | ------------------- | --------------------------------------- |
        | `def` keyword       | Used to define a function               |
        | `return` keyword    | Sends result back                       |
        | `*args`, `**kwargs` | Handle variable arguments               |
        | `lambda`            | Creates small anonymous functions       |
        | Functions help you  | Reuse, organize, and simplify your code |
